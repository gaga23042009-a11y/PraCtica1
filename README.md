# PraCtica1

Telegram-бот собирает ежедневные данные о настроении, продуктивности и сне, хранит их в SQLite и показывает статистику, инсайты и графики.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python bot.py
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и инструкция |
| `/add` | Пошаговый ввод данных за сегодня |
| `/stats` | Статистика за неделю/месяц, инсайты, график |
| `/history` | Последние 7 записей |
| `/settings` | Время ежедневного напоминания (HH:MM) |
| `/clear` | Удаление всех записей (с подтверждением) |
| `/help` | Краткая справка |

Главное меню с кнопками: «Записать день», «Статистика», «История», «Настройки».

## Структура проекта

- `bot.py` — точка входа, команды и FSM
- `db_handler.py` — работа с SQLite
- `analyzer.py` — статистика и графики matplotlib
- `schema.sql` — схема таблиц
- `test_data.sql` — тестовые записи (user_id = 123456789)
